Jeux

Développer un jeu est une bonne façon pour moi d'apprendre une nouvelle technologie. C'est un defi stimulant qui pousse à la réflexion et comprend souvent une portion algorithme pour définir une victoire. Aussi, c'est un bon exemple de projet où le UI et le UX sont aussi important à considérer que le code derrière.

Tic Tac Toe

Application du classique jeu de Tic Tac Toe comprenant des fonctionnalités supplémentaires, comme la possibilité de jouer jusqu'à 4 joueurs, de nommer les joueurs et le suivis des statistiques. La grille de jeu est modifiable, allant du classique 3x3, jusqu'à 10x10.

Ce projet a été développé avec la version pré-compilé de Vue.js 2. Il a été mon projet inaugurale sous ce framework.
